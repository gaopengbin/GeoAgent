## 风险与应对

| 风险 | 概率 | 影响 | 应对 |
|------|------|------|------|
| LLM工具调用不准确 | 高 | 分析失败 | 完善Few-shot示例，增加校验 |
| OSM Overpass API限流 | 中 | 数据获取慢 | 结果缓存 + 降级到天地图 |
| 大数据量浏览器卡顿 | 中 | 体验差 | 自动简化 + 聚类 + Primitive |
| Cesium热力图实现复杂 | 中 | 功能缺失 | 降级为点密度图 |
| PostGIS部署 | 低 | 初始化略慢 | Docker Compose 一键启动，开发环境零配置 |
